## Global AGENTS Template

把这份文件复制到你自己的全局规则位置后，再做替换。

### 基础占位符

- `<YOUR_CODEX_HOME>`
- `<YOUR_BACKUP_REPO_URL>`

### 全局默认要求

- 新项目默认采用双目录：`<YOUR_PROJECT>-old` / `<YOUR_PROJECT>-new`
- 每轮进入开发前，先把 `-new` 的完整基线复制到 `-old`
- 所有实际开发只在 `-new`
- 新项目或新阶段先走 PRD -> 设计文档 -> 技术文档
- 实施计划、汇报和风险说明默认使用通俗中文
- 中大任务默认采用“主代理编排，子代理执行”

### 子代理默认规则

- 默认并发 `2`
- 仅在安全条件满足时短时升到 `3`
- 首次 `429`：等待后重试一次
- 第二次 `429`：降为串行

### 规则备份

- 工作流文件更新后，可同步到 `<YOUR_BACKUP_REPO_URL>`
