# Skillify 示例 UI 设计文档

## 1. 设计基调

Skillify 的体验分成两段：

- 首页：情绪化、故事化、像产品演示页
- 登录后区域：理性、专业、可信

这两段体验必须明显区分，但仍属于同一品牌体系。

## 2. 首页方向

- 纵向整屏滚动
- 类似 PPT 的分屏叙事
- 用大标题、说明文案和连续故事建立购买兴趣
- 不让复杂表单打断浏览

## 3. 登录后区域方向

- 信息密度更高
- 页面结构更清晰
- 方案选购页使用“主内容区 + 常驻 AI 顾问区”双栏
- 移动端中顾问区改为底部可展开抽屉

## 4. 核心页面说明

### 首页 `/`

- 目标：解释服务价值，推动用户进入登录
- CTA：注册登录、获取我的方案、查看 FAQ

### 登录 / 注册页 `/auth`

- 目标：让用户快速进入平台
- 关键模块：手机号 / 邮箱切换、验证码、发送按钮、说明卡片

### 补全联系方式页 `/complete-profile`

- 目标：完成联系信息闭环
- 关键模块：缺失联系方式说明、输入框、验证码、提交按钮

### 方案选购页 `/solutions`

- 目标：展示方案并帮助用户做决策
- 关键模块：套餐卡片、详情区、定制入口、右侧顾问区

### 自定义方案页 `/custom-plan`

- 目标：把复杂需求结构化
- 关键模块：表单输入区、顾问摘要区、支付说明区

### 支付与订单页

- 支付确认页负责展示订单摘要和支付方式
- 支付结果页负责告知结果和下一步动作
- 订单页负责回看支付状态和服务状态

## 5. 关键交互

- 登录方式切换
- 验证码倒计时
- 顾问推荐与快捷问题
- 支付成功 / 失败 / 取消反馈
- 订单状态统一展示
